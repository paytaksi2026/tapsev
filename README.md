
TikTok Wheel PRO v3

Deploy (Render):

1. Upload project to GitHub
2. Create Render Web Service
3. Create PostgreSQL database
4. Add ENV variables:

DATABASE_URL=postgres://...
TIKTOK_USERNAME=your_tiktok

Deploy.

Features:
- PostgreSQL persistent queue
- Auto DB table creation
- TikTok auto reconnect
- Queue spin system
